# Methodology Notes (Option B)

- Focused on high-value conceptual checks: mechanism, comparison, and practical deployment reasoning.
- Chose questions that require retrieving semantically specific evidence, not broad textbook definitions.
- Ensured source diversity: used all four videos with heavier weight on 3Blue1Brown for mechanism depth.
- Kept answer style concise and factual so this set can be reused as a grading reference.
- Added timestamps as approximate anchors to sections that typically contain the relevant explanation.

## What these questions test

- Retrieval precision for core concepts (neuron computation, activation nonlinearity).
- Retrieval quality on training intuition (loss minimization and gradient-based updates).
- Correct mechanism disambiguation (self-attention vs local/recurrent context handling).
- Cross-video synthesis (deep learning vs traditional ML feature engineering).
- Real-world reasoning retrieval (data quality and evaluation pipeline importance).

## What wrong retrieval looks like

- Generic answers that omit key mechanism details.
- Mixing up unrelated terms (e.g., attention with backprop steps).
- Hallucinated implementation claims not discussed in the source section.
- Returning motivation text when the question asks for mechanism.
